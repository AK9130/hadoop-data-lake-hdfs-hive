1) Project Name
Hadoop Data Lake using HDFS and Hive

2) Objective
To store large-scale transaction data in HDFS and perform analysis using Hive.

3) Technologies Used
Hadoop HDFS 
Hive 
Linux 
CSV Dataset

4) Dataset
Bank Transaction Dataset (CSV file)

5) Architecture Flow
CSV Dataset --> HDFS Storage --> Hive External Table --> Hive Queries --> Data Analysis

6) Analysis Performed
Transaction type wise count 
Branch wise transactions 
Transaction type wise total amount 
Branch wise total amount 
Top 10 highest transactions

7) Outcome
Efficient storage and analysis of large transaction data using Hadoop ecosystem.
